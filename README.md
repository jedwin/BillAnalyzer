# BillAnalyzer (全能账单透视)

BillAnalyzer 是一个基于 React 和 Vite 构建的现代化账单分析工具，旨在帮助用户轻松统计和通过可视化图表了解自己的消费情况。它支持导入微信支付和支付宝的账单文件，提供自动编码识别、多文件合并、以及多维度的收支分析。

## ✨ 主要功能

- **多平台支持**：兼容微信支付 .csv 和支付宝 .csv/.xlsx 账单格式。
- **智能解析**：
    - 自动识别文件编码（GBK / UTF-8），无需手动转换。
    - 自动合并多个账单文件，去除重复交易。
    - 智能识别表头和数据列。
- **数据可视化**：
    - **收支概览**：直观展示总收入、总支出及结余。
    - **趋势分析**：支持按日、按月、按年查看收支趋势图。
    - **分类统计**：按交易类型、商户或支付方式进行饼图/条形图分析。
- **交互式筛选**：
    - 支持日期范围筛选（预设或自定义）。
    - 实时搜索（按商品、交易对象、备注等）。
    - 交易详情查看。
- **隐私安全**：所有数据处理均在**本地浏览器**完成，不会上传到任何服务器。

## 🛠 技术栈

- **前端框架**: [React 18](https://react.dev/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **图表库**: [Recharts](https://recharts.org/)
- **图标库**: [Lucide React](https://lucide.dev/)
- **Excel 处理**: [SheetJS (XLSX)](https://docs.sheetjs.com/) (目前通过 CDN 动态加载)
- **样式**: Tailwind CSS (内联样式与 Utility Classes)

## 🚀 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) (推荐 v18 或更高版本)
- npm 或 yarn 包管理器

### 安装步骤

1. **克隆项目到本地**
   ```bash
   git clone https://github.com/jedwin/BillAnalyzer.git
   cd BillAnalyzer
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```
   启动后，浏览器通常会自动打开 `http://localhost:5173`。

### 构建部署

如果需要部署到生产环境（静态托管）：

```bash
npm run build
```

构建产物将输出到 `dist` 目录。你可以使用 `npm run preview` 在本地预览构建结果。

## 📝 使用指南

1. **导出账单**：
    - **微信**：打开微信 -> 我 -> 服务 -> 钱包 -> 账单 -> 常见问题 -> 下载账单 -> 用于个人对账。
    - **支付宝**：打开支付宝 -> 我的 -> 账单 -> 点击右上角“...” -> 开具交易流水证明 -> 用于个人对账。
2. **导入文件**：
    - 点击页面右上角的上传区域，或直接将解压后的 CSV/Excel 文件拖入页面。
    - 支持一次性选择多个文件。
3. **查看分析**：
    - 使用顶部的日期筛选器查看特定时间段。
    - 点击图表元素可进行下钻筛选。

## ⚠️ 注意事项

- **Excel解析依赖**：当前项目使用 CDN 加载 Excel 解析库 (`xlsx.full.min.js`)。如果在断网环境下使用，Excel 格式导入可能会失效（CSV 仍可正常工作）。
- **隐私说明**：本项目是一个纯前端应用，您的账单数据**仅在本地及内存中处理**，刷新页面后数据将清空，请放心使用。
